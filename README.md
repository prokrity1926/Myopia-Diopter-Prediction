# Myopia Diopter Prediction Using Machine Learning

## Overview
This project predicts eye power (diopter) using Machine Learning.

## Technologies Used
- Python
- Pandas
- Scikit-Learn
- Matplotlib
- Google Colab

## Model
- Random Forest Regressor

## Results
- MAE: 0.58
- R² Score: 0.44
